<h1> Download and install </h1>
<br>
- git clone https://github.com/KATalyzt75/twitchBotLVLRS <br>
- cd twitchBotLVLRS <br>
- sudo npm i -D <br>
- npm run dev <br>
<br>
<br>
<h2> Edit anything you need! </h2>
You can edit (extra): <br>
- Debug (ON / OFF) <br>
- Dictionary (For your own language) <br>
- Last line to get Stream ID (Uncomment that and send a message in the target channel!) <br>
- And any output message! (marked in the code) <br>
